・使い方

・テキストボックスのところにLaTex形式の文章を入力すると、きれいな数式を出力してくれる。
・KaTexで動いているため、一部のLatexコマンドが使えないので注意。

$a+b=c$⇒⇒⇒$$a+b=c$$

    ・式をそろえる方法
    ・alignedとcasesがある。casesは場合分けなどに使える。
    ・&をいれると、その位置で数式をそろえてくれる。
    コマンド例 結果
    aligned \begin{aligned}
    a + b &= c + d\\
    &=e + f
    \end{aligned}
    		 $$ \begin{aligned} a + b &= c + d\\ &=e + f \end{aligned} $$
    cases f(x)=
    \begin{cases}
    a + b\\c
    \end{cases}
    		 $$ f(x)=\begin{cases} a + b\\ c \end{cases} $$

    ・LaTexでは\begin{eqnarray}など様々な数式モードがあるが、KaTexでは\begin{aligned}しか使えないので注意。
